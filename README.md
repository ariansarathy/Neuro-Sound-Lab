# 🧠 Neuro-Sound Lab
Neuro-Sound Lab is an experimental, neuroscience-inspired web application that generates personalized soundscapes based on a user’s cognitive and emotional state. Rather than recommending existing music, the platform synthesizes audio environments designed to align with focus, stress, energy, and mood levels.
The project explores the intersection of music, neuroscience, and web technology, using science-informed heuristics to create explainable, adaptive sound experiences.

## 🔬 How It Works
### Cognitive & Mood Assessment
Users complete a short, non-clinical assessment measuring factors such as focus, stress, energy, and emotional tone.
### State Analysis Engine
Responses are converted into weighted cognitive scores using neuroscience-inspired logic.
### Soundscape Synthesis
The system generates layered audio using the Web Audio API, adjusting frequency ranges, tempo, spatial depth, and texture based on the user’s mental state.
### Neuroscience Explanation
Each soundscape includes a brief, educational explanation describing how specific audio characteristics may influence cognition and perception.

## 🎧 Core Features
- Personalized soundscape generation
- Cognitive & emotional state modeling
- Real-time audio synthesis
- Science-forward explanations (non-medical)
- Minimal, lab-style UI design

## 🧪 Project Goals
- Explore how sound can influence cognitive states
- Build explainable, non-black-box audio experiences
- Combine neuroscience concepts with interactive web design
- Create a unique, research-inspired portfolio project

## ⚠️ Disclaimer
Neuro-Sound Lab is for educational and experimental purposes only and is not intended for medical diagnosis or treatment.
